# Backend Laravel (Vazio)

Este diretório contém a estrutura de pastas para o backend Laravel.

As pastas estão vazias e precisam ser preenchidas com o código da API.

## Estrutura esperada

```
backend/
├── app/                 # Modelos, Controllers, Middlewares
├── bootstrap/          # Arquivos de bootstrap do Laravel
├── config/             # Configurações (database, auth, cache, etc)
├── database/           # Migrations, seeders, factories
├── public/             # Ponto de entrada (index.php)
├── resources/          # Views, CSS, JS
├── routes/             # Definição de rotas (api.php, web.php)
├── storage/            # Logs, cache, uploads
└── tests/              # Testes unitários e feature
```

Adicione seu código Laravel neste diretório seguindo a estrutura acima.
